# Data Inventory

## Project
South African Healthcare Access Analysis

## Objective
To evaluate healthcare facility distribution across South Africa and assess access relative to population distribution.

## Planned Datasets

| Dataset | Source | Format | Status |
|----------|----------|----------|----------|
| Healthcare Facilities Register | National Department of Health | CSV/XLSX | Pending |
| Provincial Population Estimates | Statistics South Africa | CSV/XLSX | Pending |
| Provincial Boundaries | Municipal Demarcation Board | CSV/Shapefile | Pending |

## Key Variables of Interest

### Healthcare Facilities
- Facility Name
- Facility Type
- Province
- District
- Municipality
- Latitude
- Longitude

### Population Data
- Province
- Population
- Population Density
- Urban/Rural Classification

### Geographic Data
- Province Code
- District Code
- Boundary Geometry

## Expected Outputs

- Facility density by province
- Facilities per 100,000 population
- Geographic distribution maps
- Provincial healthcare access rankings
- Healthcare infrastructure insights
